# Stielejtes integrál
$$\int_a^b f(x) dg(x) = \lim_{n \to \infty} \sum_{k=1}^n f(x_k)[g(z_k) - g(z_{k1})]$$
A következő tulajdonságokkal rendelkezik:
- $\int_a^b f(x)dg(x) + \int_a^b g(x)df(x) = [f(x) g(x)]_a^b$
- $\int_a^b f(x)dg(x) = \int_a^b f(x) g^\prime (x) dx$
